About

BackVision is a cutting-edge video background and media slider plugin. You can use it to loop a single background video or create impressive video/image gallery with fullscreen mode and time controlled banners. It's responsive, fast and flexible!

BackVision uses super fast video processing based on requestAnimationFrame routines, so the playback will be flawless in all modern browsers. It has a fullscreen mode, navigation controls, sound control and progress bar. You can touch-swipe/drag videos in carousel mode with cool accordion effect!

Using a banner feature you can setup banners to pop-up at specific second of a video using one of predefined animations. This way you can make your videos more impressive and meaningful.

Key Features

  • Unlimited videos in mp4/webm/ogg formats
  • Scalable and responsive design
  • Sleep mode when out if view
  • Display your logo before first video will start
  • Crisp SVG controls with ability to change their color via plugin options
  • Animated banners that you can setup to pop-up at specific second of your video
  • Accordion effect when touch-swipe/drag to next/previous video
  • Ability to set number of slices for accordion effect
  • Fallback to poster images for unsupported devices or video load error
  • Touch-drag support & navigation controls
  • Keyboard support (next/prev and pause)
  • Support for desktop & mobile devices
